Student Performance Prediction

This is my beginner machine learning project.

In this project, I tried to understand how study hours, attendance and sleep hours can affect student marks. I created a small dataset, explored it using Python and plotted some graphs to understand the data.

After that, I trained a Linear Regression model and checked its accuracy. Through this project, I learned the basic steps involved in a machine learning workflow such as data exploration, visualization, model training and evaluation.

Tools used:

- Python
- Pandas
- Matplotlib
- Scikit-learn

This project was done as part of my learning process in machine learning.
